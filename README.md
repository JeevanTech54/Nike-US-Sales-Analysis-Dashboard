# Nike-US-Sales-Analysis-Dashboard
Interactive Power BI dashboard analyzing Nike sales performance across regions, products, and time. It highlights revenue trends, top-selling categories, and customer demand, enabling data-driven business insights through dynamic visuals and KPIs.
Nike US Sales Dashboard (Power BI)

Project Overview
Developed an interactive Power BI dashboard to analyze Nike’s US sales data and uncover actionable business insights. The dashboard enables efficient monitoring of sales performance, regional trends, and product-level contributions.

Objectives
Evaluate overall sales and profitability
Identify high-performing regions and states
Analyze product category performance
Track key business metrics over time

Dataset
The dataset includes sales transactions across the United States with the following attributes:
Region
State
Product Category
Sales
Profit
Units Sold
Date

Dashboard Features
Interactive dashboard with dynamic filtering capabilities
Key performance indicators including total sales, profit, and units sold
Regional and state-level performance analysis
Category-wise sales and profitability insights
Time-series analysis for trend identification

Tools and Technologies
Power BI for data visualization
Power Query for data cleaning and transformation
DAX for creating calculated measures and KPIs

Key Insights
Discovered top-performing regions driving the majority of revenue
Identified trends and seasonal patterns in sales performance
Highlighted the most profitable product categories
Provided comparative insights across regions to support decision-making

How to Use
Download the Power BI (.pbix) file from the repository
Open the file using Power BI Desktop
Interact with filters and visuals to explore the data
